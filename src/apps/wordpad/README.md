# WordPad

## Purpose

**WordPad** is a rich text editor for azOS Second Edition. It provides more advanced formatting options than Notepad, making it suitable for creating well-formatted documents, notes, and letters.

## Key Features

- **Rich Text Editing**: Support for various fonts, sizes, and styles (Bold, Italic, Underline).
- **Alignment Options**: Left, center, and right alignment for paragraphs.
- **Lists**: Create bulleted lists for better organization.
- **Color Support**: Change text and background colors.
- **File Operations**: Standard New, Open, and Save functionality.

## How to Use

1.  Launch **WordPad** from the Start Menu.
2.  Type and format your text using the toolbar options.
3.  Use the `File` menu to save your work to the virtual file system.

## Technologies Used

- **JavaScript/HTML/CSS**: Custom implementation of a rich text editor.
- **os-gui**: Used for the application window and menus.
