# DOS Games Downloader

DOS Games Downloader is a utility that allows you to search for and download classic DOS games from Archive.org's software library.

## How to Install a Game

1.  Open **DOS Games Downloader** from the Start menu (Programs > DOS Games Downloader).
2.  In the search bar, type the name of the game you are looking for and press **Enter** or click **Search**.
3.  A list of results will appear. Find the game you want and click the **Install** button.
4.  The application will download the game ZIP file and extract it to the system.
5.  Once the installation is complete, a dialog window will appear confirming the success and showing the directory where the game was installed (e.g., `/C:/Games/[game-id]`).

## How to Play an Installed Game

After a game is successfully installed, you can play it by following these steps:

1.  Open **Windows Explorer** (or **My Computer**).
2.  Navigate to the installation directory mentioned in the success dialog (usually `/C:/Games/[game-id]`).
3.  Locate the main executable file of the game (look for `.EXE`, `.COM`, or `.BAT` files).
4.  Double-click the executable file.
5.  The system will automatically launch the game using the integrated **DOSBox** emulator.

Alternatively, you can manually create a shortcut to the game executable on your Desktop for quicker access.
