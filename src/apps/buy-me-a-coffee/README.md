# Support the Project

## Purpose

The **Buy me a coffee** app provides a direct way for users to support the development of azOS Second Edition. It integrates a support widget, making it easy to contribute to the project's ongoing maintenance and feature development.

## Key Features

- **Direct Integration**: Seamlessly embeds the Ko-fi support widget within a system window.
- **Easy Contribution**: Allows for quick donations and messages to the developer.
- **Secure**: Uses the official Ko-fi widget for secure transactions.

## How to Use

1.  Launch **Buy me a coffee** from the desktop.
2.  The Ko-fi widget will load within the window.
3.  Follow the on-screen instructions to make a contribution or send a message.

## Technologies Used

- **Ko-fi Widget**: Integrated via iframe for secure payments.
- **IFrameApplication**: Managed as a singleton system application.
