# Disk Defragmenter

## Purpose

**Disk Defragmenter** is a nostalgic simulation of the classic Windows 98 utility. While it doesn't perform any actual file system optimization, it faithfully recreates the mesmerizing visual experience and mechanical sounds of a hard drive being defragmented.

## Key Features

- **Dynamic Visualization**: Generates a randomized map of "fragmented" disk sectors on launch.
- **Authentic Animation**: Watch as the system systematically reorganizes the blocks, complete with color-coded sector types (Reading, Writing, Optimized, etc.).
- **Immersive Sound**: Plays a loop of authentic hard drive mechanical sounds during the process.
- **Playback Controls**: Pause, resume, or stop the defragmentation process at any time.

## How to Use

1.  Launch **Disk Defragmenter** from the Start Menu.
2.  Click **Defragment** to start the simulation.
3.  Click **Details** to see the legendary grid view of disk clusters.
4.  Sit back and enjoy the nostalgia. Note that closing the app resets the "disk" state.

## Technologies Used

- **JavaScript/Canvas**: For the high-performance rendering of the cluster grid and animations.
- **System Sound Manager**: Plays the authentic hard drive loop.
